support natively, built by reverse-engineering its undocumented internal API. Diagnoses
and works around an anti-bot challenge, discovers an undocumented data schema empirically,
and is engineered for resilience and responsible request behavior (caching, pacing,
pagination, sort rotation). A personal/demo project, intentionally kept to light use.
