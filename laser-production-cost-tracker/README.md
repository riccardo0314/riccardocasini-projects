A lightweight, self-hosted MES that automatically collects production data from two
industrial laser cutting machines and computes the cost of each job from its actual run
time. Built to replace a discontinued commercial system, across legacy Windows 7 machines
on a factory LAN , including one where Python couldn't be installed at all. Event-driven
data capture, append-only storage with built-in data integrity, and a read-only reporting
UI. Deployed and running in production at zero software-licensing cost.
